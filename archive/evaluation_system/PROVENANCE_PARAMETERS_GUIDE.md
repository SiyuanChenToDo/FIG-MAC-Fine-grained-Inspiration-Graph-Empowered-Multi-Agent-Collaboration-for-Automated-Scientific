
# 指标计算参数说明

## 溯源指标参数 (Provenance Metrics Parameters)

### α (alpha) - 源相似度权重
- **默认值**: 0.5
- **范围**: [0, 1]
- **含义**: 控制源相似度对溯源因子的影响
- **解释**:
  - α = 0: 完全忽略源相似度，只考虑源多样性
  - α = 0.5: 平衡考虑源相似度和源多样性
  - α = 1: 完全依赖源相似度，忽略源多样性
- **推荐值**: 0.5（平衡）

### β (beta) - 源多样性权重
- **默认值**: 0.5
- **范围**: [0, 1]
- **含义**: 控制源多样性对溯源因子的影响
- **解释**:
  - β = 0: 完全忽略源多样性，只考虑源相似度
  - β = 0.5: 平衡考虑源相似度和源多样性
  - β = 1: 完全依赖源多样性，忽略源相似度
- **推荐值**: 0.5（平衡）
- **约束**: α + β 不一定等于1，但通常设为相等

### γ (gamma) - 溯源因子权重
- **默认值**: 0.7
- **范围**: [0, 1]
- **含义**: 控制溯源因子G对最终P指标的影响
- **公式**: P = ON_raw × (γG + (1-γ))
- **解释**:
  - γ = 0: P = ON_raw，完全忽略溯源因子
  - γ = 0.5: P = ON_raw × (0.5G + 0.5)，平衡考虑
  - γ = 1: P = ON_raw × G，完全依赖溯源因子
- **推荐值**: 0.7（更重视溯源因子）

## 新颖性指标参数

### ON_v2 归一化公式
- **原公式**: ON = rank / (N-1)，范围 [0, 1]
- **改进公式**: ON = (rank + 1) / N，范围 [1/N, 1]
- **改进原因**: 避免最低排名得到0，所有报告都得到非零值

## 参数调整建议

### 场景1：强调溯源重要性
```
alpha = 0.5, beta = 0.5, gamma = 0.8
```
- 提高γ到0.8，使溯源因子影响更大

### 场景2：强调源多样性
```
alpha = 0.3, beta = 0.7, gamma = 0.7
```
- 降低α，提高β，更重视源多样性

### 场景3：强调源相似度（避免重复）
```
alpha = 0.7, beta = 0.3, gamma = 0.7
```
- 提高α，降低β，更重视低相似度（避免重复）

### 场景4：完全忽略溯源因子
```
gamma = 0.0
```
- 此时P = ON_raw，溯源因子无影响
