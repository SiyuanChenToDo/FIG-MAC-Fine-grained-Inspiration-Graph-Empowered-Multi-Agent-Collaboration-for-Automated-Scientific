\section{Introduction}

These instructions are for authors submitting papers to *ACL conferences using \LaTeX. They are not self-contained. All authors must follow the general instructions for *ACL proceedings,\footnote{\url{http://acl-org.github.io/ACLPUB/formatting.html}} and this document contains additional instructions for the \LaTeX{} style files.

The templates include the \LaTeX{} source of this document (\texttt{acl\_latex.tex}),
the \LaTeX{} style file used to format it (\texttt{acl.sty}),
an ACL bibliography style (\texttt{acl\_natbib.bst}),
an example bibliography (\texttt{custom.bib}),
and the bibliography for the ACL Anthology (\texttt{anthology.bib}).
% 歇后语是一种常见的中文语言现象，通常由一个谜面和一个谜底构成。如图1所示，歇后语的深层含义往往不直接显现于字面，其前后两部分之间存在着逻辑推理的联系，且伴随着谐音双关语的使用。因此，对于机器尤其是大型语言模型而言，准确把握歇后语的深层含义是一项重大挑战。深入理解歇后语背后的隐喻语义和探索其复杂的推理模式，不仅能够有效评估和提升现有大型语言模型对歇后语的理解能力，还能进一步推动文本生成、机器翻译等下游任务的性能。

% 为了解决以上挑战，我们首先调研了现有的阅读理解工作，这些工作涉及理解通用语义的中文通用表达数据集（d1, d2, d3），专注于隐喻理解的成语数据集（d4, d5, d6），以及识别古汉语语义的古典中文数据集（d7, d8, d9）。在通用表达领域，Cui (2016)、QLSC (2023) 和 ddk (2024) 的研究通过共识注意力机制、潜在特征嵌入和模型蒸馏技术，提高了模型在多场景下的语义理解能力。在中文成语领域，Long (2021) 和 Wu (2024) 的研究利用图神经网络和混合自注意力机制的对比学习方法，增强了模型对隐喻的理解。在古典中文领域，CLT (2022) 和 Liu (2024) 提出的对比学习和跨引导方法，提升了模型对整体语义的理解性能。现有阅读理解工作有效促进了模型多场景语义理解和隐喻理解的能力。

% 然而，当前的阅读理解研究主要集中于传统句子的语义理解，但对于中文中独特的知识推理和语言现象的深入探索仍有所欠缺。中文歇后语以其独特的形式和复杂的推理机制，成为了理解中文背景下知识关联推理、隐喻表达和双关表达问题的关键。因此，为了深入理解歇后语的深层含义，我们借鉴了先前的研究工作，如【sun 2019】【wu 2024】，将歇后语的理解任务转化为歇后语完形填空任务。歇后语完形填空任务的核心目标是使机器在阅读包含歇后语的文本后，能够从候选集中选取最合适的选项填充空白处。根据任务的性质，我们归纳了歇后语完形填空任务面临的三大挑战：（1）知识推理：歇后语中的谜题和谜底之间存在逻辑推理关系，这要求我们理解并捕捉这种内在联系。（2）隐喻表达：歇后语中常常存在字面意义与实际含义不一致的情况，这要求我们深入挖掘歇后语的实际语义。（3）双关表达：歇后语中的双关表达主要表现为谐音双关和语义双关两种形式，分别由中文的同音字和一词多义现象引起，这要求我们识别并理解这些双关表达所引发的语义变化。

% 为了解决上述挑战，我们首先构建了一个歇后语阅读理解数据集（CHTD）。数据来源于知乎、微博以及歇后语词典，经过人工标注，我们共收集了18103个歇后语及其对应的75296个文本。CHTD数据集覆盖了日常对话、语言考试、文学作品和新闻文刊等多个领域。

% 基于CHTD数据集，我们提出了一种多视角特征的歇后语对比学习网络（MTCLN）。该网络通过联合交叉投影的方法，获取候选歇后语的个性和共性表达，以区分近义选项。此外，我们采用了软化对比学习方法来处理前后知识推理和多对多的关系。最后，我们利用多特征注意力融合技术，结合拼音向量进行注意力级别的融合，以解决双关语问题。在CHTD数据集上进行的广泛实验验证了我们方法的有效性，实现了在四个测试集中56%的平均准确率。


\section{Related Work}
Existing metaphor research is mainly based on selection preference violation (SPV) \cite{wilks1975preferential,wilks1978making}, metaphor identification procedure (MIP) \cite{group2007mip}, and conceptual metaphor theory (CMT) \cite{Lakoff1980-LAKMWL}. \cite{yang2024can} Use SPV theory and unsupervised methods to improve metaphor detection; \cite{mao2024metapro} Combine MIP and semantic contrast methods to improve metaphor recognition; \cite{li2024embodiedbert} Propose the EmbodiedBERT model that integrates SPV and MIP to enhance metaphor understanding; \cite{jia2024metaphor} Propose the MiceCL model that integrates SPV and MIP to alleviate the data sparsity problem in metaphor detection; \cite{tian2024bridging} Develop an explainable metaphor detection method based on CMT theory to achieve fine-grained verb metaphor detection; \cite{tong2024metaphor} Explore the performance of LLMs in generating metaphorical interpretations of words; \cite{tian2024theory} Optimize metaphor detection with the help of CMT, MIP, SPV theory and knowledge graphs; \cite{wang2024mmte} Analyze the impact of metaphors on machine translation in cross-language scenarios; \cite{chen2024merely} Analyze the challenges of metaphor understanding through LLMs prompts; \cite{ahmed2024generating} Use sentence metaphorical transformation to increase the difficulty in the cross-document event coreference resolution task. In addition, research on metaphor in areas such as advertising, news editorials, image generation and understanding is also advancing \cite{xu2024exploringMetaphor,sengupta2024analyzing,zhang2024gome,zhang2024camel}.

To produce a PDF file, pdf\LaTeX{} is strongly recommended (over original \LaTeX{} plus dvips+ps2pdf or dvipdf). Xe\LaTeX{} also produces PDF files, and is especially suitable for text in non-Latin scripts.

\section{Dataset Construction}

The first line of the file must be
\begin{quote}
\begin{verbatim}
\documentclass[11pt]{article}
\end{verbatim}
\end{quote}

To load the style file in the review version:
\begin{quote}
\begin{verbatim}
\usepackage[review]{acl}
\end{verbatim}
\end{quote}
For the final version, omit the \verb|review| option:
\begin{quote}
\begin{verbatim}
\usepackage{acl}
\end{verbatim}
\end{quote}

To use Times Roman, put the following in the preamble:
\begin{quote}
\begin{verbatim}
\usepackage{times}
\end{verbatim}
\end{quote}
(Alternatives like txfonts or newtx are also acceptable.)

Please see the \LaTeX{} source of this document for comments on other packages that may be useful.

Set the title and author using \verb|\title| and \verb|\author|. Within the author list, format multiple authors using \verb|\and| and \verb|\And| and \verb|\AND|; please see the \LaTeX{} source for examples.

By default, the box containing the title and author names is set to the minimum of 5 cm. If you need more space, include the following in the preamble:
\begin{quote}
\begin{verbatim}
\setlength\titlebox{<dim>}
\end{verbatim}
\end{quote}
where \verb|<dim>| is replaced with a length. Do not set this length smaller than 5 cm.

\subsection{Hyperlinks}

Users of older versions of \LaTeX{} may encounter the following error during compilation:
\begin{quote}
\verb|\pdfendlink| ended up in different nesting level than \verb|\pdfstartlink|.
\end{quote}
This happens when pdf\LaTeX{} is used and a citation splits across a page boundary. The best way to fix this is to upgrade \LaTeX{} to 2018-12-01 or later.

Table~\ref{citation-guide} shows the syntax supported by the style files.
We encourage you to use the natbib styles.
You can use the command \verb|\citet| (cite in text) to get ``author (year)'' citations, like this citation to a paper by \citet{Gusfield:97}.
You can use the command \verb|\citep| (cite in parentheses) to get ``(author, year)'' citations \citep{Gusfield:97}.
You can use the command \verb|\citealp| (alternative cite without parentheses) to get ``author, year'' citations, which is useful for using citations within parentheses (e.g. \citealp{Gusfield:97}).

A possessive citation can be made with the command \verb|\citeposs|.
This is not a standard natbib command, so it is generally not compatible
with other style files.

\section{Bib\TeX{} Files}
\label{sec:bibtex}

Unicode cannot be used in Bib\TeX{} entries, and some ways of typing special characters can disrupt Bib\TeX's alphabetization. The recommended way of typing special characters is shown in Table~\ref{tab:accents}.

Please ensure that Bib\TeX{} records contain DOIs or URLs when possible, and for all the ACL materials that you reference.
Use the \verb|doi| field for DOIs and the \verb|url| field for URLs.
If a Bib\TeX{} entry has a URL or DOI field, the paper title in the references section will appear as a hyperlink to the paper, using the hyperref \LaTeX{} package.
