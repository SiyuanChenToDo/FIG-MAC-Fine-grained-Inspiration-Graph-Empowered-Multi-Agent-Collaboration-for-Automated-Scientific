# 批量重新计算命令（使用正确的报告目录）

## 报告目录
`/root/autodl-tmp/Myexamples/evaluation_system/batch_results/ours/reports`

## 方案1：使用 aggregate_metrics_to_excel.py（推荐）

### 优点
- ✅ 直接从报告目录读取，更可靠
- ✅ 支持命令行参数，灵活配置
- ✅ 自动ON归一化
- ✅ 生成全新的干净数据

### 命令

```bash
cd /root/autodl-tmp

# 小批量测试（先处理5个报告，检查结果）
python Myexamples/evaluation_system/aggregate_metrics_to_excel.py \
    --report-dir Myexamples/evaluation_system/batch_results/ours/reports \
    --output-excel Myexamples/evaluation_system/batch_results/ours/metrics_summary_test_5.xlsx \
    --limit 5

# 或者处理10个报告
python Myexamples/evaluation_system/aggregate_metrics_to_excel.py \
    --report-dir Myexamples/evaluation_system/batch_results/ours/reports \
    --output-excel Myexamples/evaluation_system/batch_results/ours/metrics_summary_test_10.xlsx \
    --limit 10

# 完整运行（处理所有150个报告）
python Myexamples/evaluation_system/aggregate_metrics_to_excel.py \
    --report-dir Myexamples/evaluation_system/batch_results/ours/reports \
    --output-excel Myexamples/evaluation_system/batch_results/ours/metrics_summary_improved.xlsx

# 后台运行（推荐，因为需要较长时间）
nohup python Myexamples/evaluation_system/aggregate_metrics_to_excel.py \
    --report-dir Myexamples/evaluation_system/batch_results/ours/reports \
    --output-excel Myexamples/evaluation_system/batch_results/ours/metrics_summary_improved.xlsx \
    > aggregate_improved.log 2>&1 &

# 查看进度
tail -f aggregate_improved.log

# 检查进程
ps aux | grep aggregate_metrics_to_excel
```

## 推荐方案

### 🏆 推荐使用：方案1（aggregate_metrics_to_excel.py）

**原因**：
1. 更可靠：直接从报告目录读取，不依赖Excel中的路径
2. 更灵活：支持命令行参数
3. 更完整：自动处理ON归一化
4. 更干净：生成全新的数据文件

**执行命令**：

### 步骤1：小批量测试（推荐先运行）

```bash
cd /root/autodl-tmp

# 先测试5个报告（约5-10分钟）
python Myexamples/evaluation_system/aggregate_metrics_to_excel.py \
    --report-dir Myexamples/evaluation_system/batch_results/ours/reports \
    --output-excel Myexamples/evaluation_system/batch_results/ours/metrics_summary_test_5.xlsx \
    --limit 5

# 检查结果
python -c "
import pandas as pd
df = pd.read_excel('Myexamples/evaluation_system/batch_results/ours/metrics_summary_test_5.xlsx')
print(f'处理了 {len(df)} 个报告')
print('\n关键指标平均值:')
for col in ['objective.Fluency_Score', 'subjective_llm.Novelty', 'subjective_llm.Significance', 'subjective_llm.Effectiveness']:
    if col in df.columns:
        val = pd.to_numeric(df[col], errors='coerce').mean()
        print(f'  {col}: {val:.4f}')
"
```

### 步骤2：如果结果满意，运行完整批量

```bash
cd /root/autodl-tmp

# 后台运行所有150个报告

    --output-excel Myexamples/evaluation_system/batch_results/ours/metrics_summary_improved.xlsx \
    > aggregate_improved.log 2>&1 &

# 查看实时进度
tail -f aggregate_improved.log
```
